# Loop Community

**Stay in a productive loop** - Open source gesture-based task management

Loop Community is the free, open-source version of Loop, featuring core productivity tools with gesture-based interactions and local-first data storage.

- **Gesture-Based UI**: Swipe to edit, delete, and manage tasks
